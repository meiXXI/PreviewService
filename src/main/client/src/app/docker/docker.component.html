Run the Preview Service locally on your machine

The Preview Service has been developed as a Docker Container. This means, it can be seen as a standardized piece of software which is executable on may systems out-of-the-box - also on your local machine. The only prerequisite is the docker container runtime installed on your system. More details about the installation can be found here: https://docs.docker.com/install/.

The Docker Container Image of the PreviewService is available on Docker Hub: https://hub.docker.com/r/ricebean/preview-service.
