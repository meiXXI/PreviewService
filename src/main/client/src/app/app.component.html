    Introduction

    This Preview Service allows the generation of preview images from the first page of PDF Documents. The preview image can be either generated manually, using this graphical user interface, or automatically, using the API as described in below.

    The Preview Service has been implemented as a proof of concept for evalution porposes of a publication about how to use cloud computing technologies in the printing industry.




    Manual Preview Generation

    This section is for the manual generation of preview images using the Preview Service.




    Automated Preview Generation (API)

    The section provides information and materials of how to generate preview images using the XJDF (Exchange Job Definition Format) based API (Application Programming Interface) of the Preview Service.




    Server Logging

    The logging section aims to make the internal processes of the Preview Service more transparent to the user.
